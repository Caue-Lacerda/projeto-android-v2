Segunda Versão do Projeto Android

O Projeto Android foi desenvolvido e criado pelo professor Gustavo Guanabara para o canal CursoemVídeo. Este site não tem o objetivo de tirar vantagem do trabalho de Gustavo, pelo contrário, esta segunda versão é uma forma de provar para mim mesmo que consigo criar sites utilizando HTML, CSS e JavaScript, ou seja, o "sor", como prefiro dizer, aconselha nos vídeos que façamos várias versões do código para desenvolver as habilidades necessárias para a criação e o desenvolvimento de sites.

Agradecimentos

Queria deixar meus agradecimentos ao professor Gustavo Guanabara, sem os ensinamentos dele eu não chegaria nem perto de criar um site, e é claro para quem realmente quer aprender a desenvolver sites, recomendo muito o CursoemVídeo, o sor vai te pegar pela mão e vai ensinar tudo o que você precisa saber para construir seus sites e caso algum dia o Gustavo veja isso, Obrigadão Sor! ;)
